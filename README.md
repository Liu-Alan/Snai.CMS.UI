# Snai.CMS.UI
### 内容管理UI
    技术栈：Vue3，Ant Design Vue，Axios，分页  

### 功能  
    登录，登出，修改密码(已完成)  
    用户管理(用户列表、添、删、改、禁/启用、解锁)(已完成)  
    模块管理(模块列表、添、删、改、禁/启用)(已完成)  
    角色管理(角色列表、添、删、改、禁/启用、分配权限)(已完成)  
    
### 菜单层级
    系统管理  
    -------账号管理  
    ----------------添/删/改等账号 
    
** 对应后端仓库：** [https://github.com/Liu-Alan/Snai.CMS.Api](https://github.com/Liu-Alan/Snai.CMS.Api)
